# Лабораторийн ажил: Git Clone, Commit, Push, Pull ашиглах

## 1. Түлхүүр үг
Git, GitHub, repository, clone, commit, push, pull, remote repository, local repository, version control

## 2. Хичээлийн зорилго
GitHub дээрх repository-г локал компьютерт clone хийх, өөрчлөлт хийх, commit хийх, push хийх болон pull хийх процессыг ойлгох.

## 3. Суралцах үр дүн
- GitHub repository-г clone хийх
- Локал дээр файл өөрчлөх
- git add болон git commit ашиглах
- git push ашиглан GitHub руу илгээх
- git pull ашиглан өөрчлөлт татах
